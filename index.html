Mapa bivariado: variable A y variable B por región.

Cómo leer este mapa

Cada región combina dos dimensiones:

El color — ¿en qué dirección varía la variable B?

█ Teal → la variable B disminuye

█ Magenta → la variable B aumenta

█ Lila → la variación es leve o cercana a cero

La intensidad — ¿qué tan alto es el nivel de la variable A?

Más claro = menor nivel

Más oscuro = mayor nivel

Ejemplo: teal oscuro = una región con nivel alto de A, pero con mejora reciente en B.

Pasa el cursor sobre una región para ver sus valores exactos.
